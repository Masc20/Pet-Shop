Upload Directory Structure
=========================

This folder contains uploaded files for the Pawfect Pet Shop application.

Folder Structure:
- pets/       - Pet images
- products/   - Product images  
- users/      - User avatars
- temp/       - Temporary uploads

Permissions:
Set folder permissions to 755 or 777 to allow file uploads.

Security:
- Only image files should be uploaded
- File types are validated server-side
- Files are renamed with timestamps for security
